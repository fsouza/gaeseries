acquire(self, startup=False)

source code 

returns the time when the lock is acquired or None if cron already runing

lock is implemnted by writing a pickle (start, stop) in cron.master start is time when cron job starts and stop is time when cron completed stop == 0 if job started but did not yet complete if a cron job started within less than 60 secods, acquire returns None if a cron job started before 60 seconds and did not stop, a warning is issue "Stale cron.master detected"
