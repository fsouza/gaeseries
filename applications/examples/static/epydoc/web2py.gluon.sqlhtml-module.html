SQLFORM is used to map a table (and a current record) into an HTML form given a SQLTable stored in db.table generates an insert form:: SQLFORM(db.table) generates an update form:: record=db(db.table.id==some_id).select()[0] SQLFORM(db.table, record) generates an update with a delete button:: SQLFORM(db.table, record, deletable=True) if record is an int:: record=db(db.table.id==record).select()[0] optional arguments: :param fields: a list of fields that should be placed in the form, default is all.
