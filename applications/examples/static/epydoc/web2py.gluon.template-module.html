__re_strings(text)

source code 
expose re_strings regex for doctest 
>>> __re_strings('x""y')
['""']
>>> __re_strings('""""""')
['""""""']
>>> __re_strings('"abc"')
['"abc"']
>>> __re_strings(r'"abc\"" "x"')
['"abc\\""', '"x"']
>>> __re_strings('"abc"xy"def"')
['"abc"', '"def"']
>>> __re_strings('"""abc""""')
['"""abc"""']
>>> __re_strings('"""a"b"c""""xy"')
['"""a"b"c"""', '"xy"']
